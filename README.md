# 🔔 홀리심볼 버프 타이머

메이플랜드(MapleStory Worlds) 홀리심볼 버프 잔여시간이 설정한 초에 도달하면 자동으로 알려주는 웹 서비스입니다.

## 🎯 특징

- **화면 공유 + OpenCV 템플릿 매칭** — 해상도별 버프 아이콘 스냅샷으로 정확한 시점 감지
- **완전 클라이언트 사이드** — 게임 데이터가 서버로 전송되지 않음
- **서버·API·설치 불필요** — 브라우저(Chrome/Edge)만 있으면 동작
- **4종 알림 사운드 + 볼륨 조절**

## 🖥️ 사용 방법

1. 현재 게임 **해상도**에 맞는 **알림 시간**(15초, 20초, 25초, 30초)을 선택
2. **화면 공유** 버튼 → "MapleStory Worlds" 창 선택
3. **탐지 시작** → 자동으로 감지 시작, 잔여시간 도달 시 알림

## 🛠️ 기술 스택

- Vanilla JavaScript (프레임워크 없음)
- OpenCV.js 4.8.0 — 템플릿 매칭 (`TM_CCOEFF_NORMED`)
- Web Audio / Notification API

## 📋 지원 해상도

| 해상도 | 알림 시간 |
|---|---|
| 1280×800/720 | 15초, 20초, 30초 |
| 1600×1024/900 | 7초, 10초, 15초, 20초, 25초, 30초 |
| 1920×1200/1080 | 15초, 20초, 25초, 30초 |
| 맥북 (레티나) | 25초 |
