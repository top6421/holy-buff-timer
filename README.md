# 🔔 홀리심볼 버프 타이머

메이플랜드(MapleStory Worlds) 홀리심볼 버프 잔여시간을 **화면 공유 기반**으로 실시간 감지하고, 만료 5초 전 알림을 제공하는 웹 서비스입니다.

## 🎯 특징

- **완전 클라이언트 사이드** — 게임 화면 데이터가 서버로 전송되지 않음
- **화면 공유 픽셀 분석만 사용** — 게임 클라이언트/메모리/패킷에 접근 안 함
- **서버·API·설치 불필요** — 브라우저만 있으면 동작
- **자동 아이콘 위치 추적** — 다른 버프 갱신으로 위치가 이동해도 대응

## 🖥️ 사용 방법

1. **화면 공유 시작** → 게임 창 선택
2. 홀리심볼 버프 사용 후 **탐지 시작** 클릭
3. 자동으로 아이콘 위치 탐지 → 잔여시간 카운트다운
4. 만료 5초 전: 비프음 + 브라우저 알림 + 화면 플래시로 경고

## 🛠️ 기술 스택

- Vanilla JavaScript (프레임워크 없음)
- OpenCV.js — 템플릿 매칭
- Tesseract.js — 숫자 OCR
- Web Audio / Notification / Document PIP API

## 📋 정책 준수

본 서비스는 **측정·알림 전용**이며 자동화 기능이 일체 없습니다. 상세 정책은 [POLICY.md](./POLICY.md) 참고.

⚠️ **주의:** 게임사 공식 허용은 아닙니다. 본인 판단 하에 사용해주세요.
